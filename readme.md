# INDEX

## Run

```bash
yarn && yarn dev
```

## What?

> - service 处理数据
> - store 存储数据（模仿 MongoDB mock schame，方便后续接入数据库)

- Site
  - 标签
  - 题目
  - 链接
  - 日期
  - 作者
  - 描述
- Site 搜索支持
  - 标签/分组归类
  - 题目/模糊搜索
  - 日期/时间排序&分组
  - 作者/分组
