路书项目

集中托管旅行路书 HTML。每一份路书使用独立目录，支持真实照片、地图标注、GeoJSON 坐标和后续复用。

新增路书的约定

把新的 HTML 放到 public/routebooks/你的路书-slug/index.html，再在 public/routebooks/manifest.json 里追加一条记录。推送到 master 后，GitHub Actions 会自动部署到 GitHub Pages。

{
  "slug": "norway-fjords-2026",
  "title": "挪威峡湾 4 日路书",
  "entry": "./norway-fjords-2026/",
  "destination": "Norway",
  "days": 4,
  "tags": ["挪威", "峡湾", "自驾"]
}
